# Privacy Policy

**Last updated:** January 2026

## Overview

Circuit Weather is an open-source web application that displays real-time weather radar for Formula 1 race circuits. We are committed to protecting your privacy.

## Data Collection

**We do not collect any personal data.**

- No user accounts or registration
- No cookies or tracking
- No analytics or telemetry
- No personal information stored

## Third-Party Services

This application uses the following third-party APIs and services:

### Jolpica F1 API
- **Purpose:** Fetches Formula 1 race schedule and circuit coordinates
- **Data sent:** None (read-only API)
- **Privacy policy:** [github.com/jolpica/jolpica-f1](https://github.com/jolpica/jolpica-f1)

### RainViewer API
- **Purpose:** Provides weather radar tile imagery
- **Data sent:** Your IP address (for rate limiting)
- **Privacy policy:** [rainviewer.com/privacy](https://www.rainviewer.com/privacy.html)

### OpenFreeMap
- **Purpose:** Provides base map tiles
- **Data sent:** Your IP address (standard web request)
- **Privacy policy:** [openfreemap.org](https://openfreemap.org)

### Cloudflare Pages
- **Purpose:** Hosts this website
- **Data sent:** Standard web request data (IP, user agent)
- **Privacy policy:** [cloudflare.com/privacypolicy](https://www.cloudflare.com/privacypolicy/)

## Local Storage

The application stores the following data locally in your browser:

- **Theme preference** (`theme`): Your selected light/dark mode
- **Unit preference** (`unit`): Your selected metric/imperial setting

This data never leaves your device and can be cleared through your browser settings.
